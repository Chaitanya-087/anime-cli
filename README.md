## binge + anime
> anime command line utility to binge anime

```sh
binime -s "<anime_name>"
```
returns list of animes to select from ; use the arrow keys to navigate to your anime ; select episode to watch from given range

```sh
binime -p
```
returns popular animes

```sh
binime -h
```
for further support
